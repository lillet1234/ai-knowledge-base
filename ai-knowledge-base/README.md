## 📂 目录结构说明

```
ai-knowledge-base/
 ├── prompts/            # AI 大脑：存储所有提示词工程文件
 ├── docs/               # 业务大脑：SOP、FAQ、营销素材、话术库
 ├── tools/              # 工具库：常用 AI 工具导航
 ├── assets/             # 素材库：图片、视频源文件 (Week 2 重点)
 ├── src/                # 代码库：计算器、小程序源码 (Week 4 重点)
 └── logs/               # 进度条：项目执行日志
```

## 🚀 如何使用
1.  **遇到业务问题**：查阅 `docs/` 下的 SOP 和 FAQ。
2.  **需要写文案**：复制 `prompts/copywriting_generator.md` 到 ChatGPT/Claude。
3.  **遇到客户刁难**：参考 `docs/sales_scripts/objection_scripts.md` 或使用 `prompts/objection_handling.md` 进行模拟演练。
