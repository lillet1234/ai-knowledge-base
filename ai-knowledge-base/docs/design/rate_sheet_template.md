# 每日行情利率表模板 (Daily Rate Sheet Template)

> **目标**: 将枯燥的数字转化为高可视化的“金融仪表盘”，建立每日行情的权威感。

---

## 📊 设计理念 (Design Philosophy)

1.  **数据可视化**: 拒绝密密麻麻的 Excel 截图。使用红绿箭头、趋势图、进度条来展示涨跌。
2.  **模块化**: 分为“国股”、“城商”、“三农”三大板块，便于阅读。
3.  **时效性**: 必须显著展示“日期”和“发布时间”，强调瞬时价格。

---

## 🛠️ 自动化工作流 (Automation Workflow)

为了实现“自动更新”，我们采用 **HTML + CSS 渲染法**：

1.  **数据源**: 每天只需在 Excel/JSON 中更新数字。
2.  **渲染层**: 使用一个固定的 HTML 模板 (`rate_sheet_wireframe.html`) 读取数据（初期可手动修改 HTML 内的数字）。
3.  **输出层**: 浏览器打开 HTML -> 截图/打印为 PDF -> 转换为图片。

---

## 📋 布局规范 (Layout Specs)

### 1. 顶部 (Header)
- **背景**: 科技蓝渐变。
- **内容**: 
    - 品牌 Logo (左上)。
    - 大标题：“今日银票转贴现行情” (居中)。
    - 动态时间：“2026.01.10 10:30 更新” (右上)。

### 2. 核心数据区 (Main Table)
- **国股大行 (Tier 1)**: 
    - 样式：金色边框，字体最大。
    - 指标：半年期利率、一年期利率、涨跌幅。
- **城商/农商 (Tier 2/3)**:
    - 样式：蓝色/绿色边框。
    - 指标：按“足年”和“半年”分类报价。

### 3. 底部 (Footer)
- **免责声明**: “市场波动快，价格仅供参考，实价以询价为准”。
- **联系方式**: 业务员微信二维码 / 电话。
