# AI 工具深度学习路线图 (AI Tools Learning Roadmap)

> **目标**: 掌握最前沿的 AI 模型与自动化工具，从“会用”进阶到“精通”，构建 DCC 项目的核心技术壁垒。

---

## 🧠 核心模型深度评测 (LLM Advanced)

### 1. GPT-4o (OpenAI)
- **定位**: 行业标杆，综合能力最强，多模态实时交互体验最佳。
- **学习重点**:
    - **Advanced Data Analysis**: 上传 Excel 销售数据，让 GPT 进行数据清洗与可视化分析。
    - **GPTs 开发**: 创建一个专属的“DCC 销售教练”，上传内部话术手册作为知识库。
    - **Voice Mode**: 练习与 AI 进行口语化的销售模拟对练。
- **资源**: [chatgpt.com](https://chatgpt.com/)

### 2. GLM-4.7 (智谱 AI)
- **定位**: 国产最强逻辑与长文本模型之一，中文语境理解极佳。
- **学习重点**:
    - **长文档分析**: 测试上传 500 页+ 的银行风控政策文档，提取关键条款。
    - **智能体构建 (GLMs)**: 学习使用 GLM 智能体中心创建“DCC 专属客服机器人”。
    - **API 对接**: 尝试调用 GLM-4 API 进行批量文本处理（如批量生成 100 条文案）。
- **资源**: [open.bigmodel.cn](https://open.bigmodel.cn/)

### 3. Claude 3.5 Sonnet (Anthropic)
- **定位**: 目前编程能力与逻辑推理最强的模型，UI/UX 友好（Artifacts 功能）。
- **学习重点**:
    - **Artifacts 实战**: 利用 Artifacts 实时预览 HTML/React 代码，开发计算器界面。
    - **System Prompt 调优**: 学习如何编写复杂的系统提示词，让 AI 角色扮演更稳定。
    - **代码辅助**: 配合 Cursor 使用，作为主要的代码生成引擎。
- **资源**: [claude.ai](https://claude.ai/)

### 4. Gemini 1.5 Pro (Google)
- **定位**: 超大上下文窗口 (2M token)，多模态处理能力强（视频/音频理解）。
- **学习重点**:
    - **视频内容分析**: 上传竞争对手的抖音视频，让 Gemini 拆解脚本结构。
    - **多模态输入**: 上传手绘草图，让 Gemini 生成前端代码。
    - **生态整合**: 探索 Google Workspace (Docs, Sheets) 的 AI 插件功能。
- **资源**: [aistudio.google.com](https://aistudio.google.com/)

---

## 🤖 智能体开发平台 (Agent Platforms)

### 5. Coze (扣子)
- **定位**: 字节跳动出品的零代码 Bot 开发平台，集成丰富的插件与工作流。
- **学习重点**:
    - **工作流 (Workflow)**: 搭建“新闻抓取 -> AI 总结 -> 推送飞书”的自动化流程。
    - **插件调用**: 在 Bot 中集成“必应搜索”、“PDF 解析”等插件增强能力。
    - **多渠道发布**: 将搭建好的 Bot 一键发布到微信公众号或飞书群。
- **资源**: [coze.cn](https://www.coze.cn/)

---

## ⚙️ 自动化工作流 (Automation)

### 6. n8n (Workflow Automation)
- **定位**: 强大的节点式自动化工作流工具，支持私有化部署，可连接 1000+ 应用。
- **学习重点**:
    - **基础概念**: 学习 Webhook, Cron, HTTP Request 节点的使用。
    - **实战场景 1 (监控)**: 监控特定公众号/网站更新，自动推送到飞书/微信。
    - **实战场景 2 (SOP)**: 收到客户表单 -> 调用 LLM 提取信息 -> 存入多维表格 -> 自动回复邮件。
    - **AI Agent**: 使用 n8n 的 AI Chain 节点构建复杂的业务处理流程。
- **资源**: [n8n.io](https://n8n.io/) | [n8n 官方文档](https://docs.n8n.io/)
