# 视觉素材库 (Assets/Visuals)

> 🚧 **建设中 (Week 2 任务)**

这里用于存放所有 AI 生成或人工设计的视觉素材。

## 📂 规划目录
- `posters/`: 朋友圈海报 (成交喜报、节日祝福)
- `logo/`: 品牌 Logo 源文件及变体
- `videos/`: 短视频成片及工程文件
- `raw/`: Midjourney/Flux 生成的原始底图

## 📏 命名规范 (Week 2 预设)
- **日期_类型_描述.jpg**
- 例如: `20260108_poster_cny_promo.jpg` (2026年1月8日_海报_春节大促)
