# AI 指令库 (Prompts)

这里存放的是项目的“数字员工”灵魂。每一个 `.md` 文件都代表一个具有特定技能的 AI Agent。

## 📁 文件列表

| 文件名 | 角色功能 | 适用场景 |
| :--- | :--- | :--- |
| **master_role.md** | **全栈专家** | 通用咨询、复杂问题拆解、宏观策略制定 |
| **copywriting_generator.md** | **金牌文案** | 朋友圈营销、节日海报文案、活动通知 |
| **objection_handling.md** | **王牌销售** | 客户谈判、价格博弈、信任建设 |

## 💡 使用技巧
1.  **直接复制**: 将 Markdown 内容完整复制给 ChatGPT/Claude/DeepSeek。
2.  **迭代优化**: 如果发现 AI 输出不够理想，请在对应文件的 `Constraint` (禁忌) 部分添加规则。
