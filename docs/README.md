# 业务知识库 (Docs)

这里是 DCC 项目的“大脑皮层”，存储了所有结构化的业务知识、营销素材和话术剧本。

## 📂 目录分类

- **`knowledge_base/` (业务基石)**
    - 存储 SOP、风控标准、FAQ 等核心业务逻辑。
    - *适用对象*: 业务员入职培训、风控审核。

- **`marketing/` (流量弹药)**
    - 存储朋友圈文案样本、海报文案、视频脚本。
    - *适用对象*: 每日运营宣发。

- **`sales_scripts/` (成交武器)**
    - 存储异议处理话术、谈判剧本、逼单技巧。
    - *适用对象*: 销售沟通、客户转化。
