# 票据风控审核标准 (Risk Control)

## ⛔ 红线标准 (直接拒收)
1.  **回头背书**: A->B->A->C，法律风险极高，原则上不收，除非提供完整贸易合同证明。
2.  **瑕疵票**:
    - 票面污损严重，遮挡关键信息（金额、账号、印章）。
    - 背书印章模糊不清，无法辨认字迹。
    - 被书人名称手写错误或涂改。
3.  **高风险承兑人**:
    - 房地产企业商票（恒大、融创等系列，除非有特殊渠道）。
    - 涉诉过多的网红票（如某些高风险集团子公司）。
    - 历史有多次拒付记录的企业。
4.  **挂失止付**: 票交所显示已挂失的票据。

---

## ⚠️ 瑕疵容忍度 (需降价或特批)
| 瑕疵类型 | 处理方式 | 备注 |
| :--- | :--- | :--- |
| **骑缝章不规范** | 扣款 50-200 元 | 只要不压字、不漏盖即可 |
| **印章轻微模糊** | 需出具《说明函》 | 需加盖公章证明 |
| **背书日期倒流** | 视情况而定 | 一般银行不收，民间资方可收但利率高 |
| **小额票据** | 加收手续费 | <5万 视为碎票，处理成本高 |

---

## 🔍 承兑人白名单 (优质资产)
1.  **国股大行**: 工、农、中、建、交、邮储（利率最低，秒贴）。
2.  **城商行 Top 10**: 北京银行、上海银行、江苏银行等。
3.  **AAA 级央企**: 中铁、中建、国网等商票（可视同银票定价）。

## 🛡️ 交易安全提示
- **视频验票**: 首次合作的大额交易，建议视频连线，确认操作人身份。
- **对公账户**: 坚持公对公打款，严禁打入个人账户（除非是个人经营贷背书）。
