# 成交喜报通用模板设计 (Success Report Templates)

> **目标**: 结合 Claude Artifacts 的代码生成能力与 Canva 的设计能力，快速产出标准化的成交喜报。

---

## 🛠️ 方法论：Claude Artifacts + Canva

1.  **Claude 布局**: 使用 AI 生成精确的 HTML/Tailwind 布局代码，确定文字层级、配色比例和留白。
2.  **Canva 渲染**: 参考 Claude 生成的布局，在 Canva 中替换为高清 3D 素材和品牌字体，导出最终图片。

---

## 📋 模板一：单笔大额成交 (The Big Deal)

### 1. 设计规范
- **核心焦点**: 数字（金额）。
- **配色**: 黑金背景 + 金色数字 + 白色说明文。
- **视觉流**: 顶部（品牌Logo） -> 中部（巨大金额） -> 底部（客户见证/信任背书）。

### 2. Claude Artifacts 提示词 (Prompt)
> 复制以下指令给 Claude，让它生成可视化布局：

```text
请使用 React + Tailwind CSS 创建一个“金融成交喜报”的手机海报布局组件 (AspectRatio 3:4)。
品牌色：主色 #0052CC (金融蓝), 辅助色 #D4AF37 (黑金)。
背景：深色渐变背景。
内容结构：
1. 顶部：居中显示 "DCC 三湘票据" Logo 和 "成交喜报" 字样。
2. 核心区：超大号金色数字 "500万"，下方配小字 "银票秒贴 | 极速到账"。
3. 详情区：一个半透明的卡片，列出 "客户：某建筑央企", "耗时：8分钟", "利率：1.65%"。
4. 底部：一条金色的 Slogan "资金方直投，无中间商赚差价"，以及二维码占位符。
要求：字体粗壮有力，具有高端商务感，带有微弱的光影质感。
```

## 📋 模板二：批量收票战报 (Volume Report)

### 1. 设计规范
- **核心焦点**: 列表感（多笔交易）。
- **配色**: 蓝色背景 + 白色卡片。
- **视觉流**: 标题（今日战绩） -> 滚动列表 -> 底部统计。

### 2. Claude Artifacts 提示词 (Prompt)
> 复制以下指令给 Claude：

```text
请使用 React + Tailwind CSS 设计一张“今日战绩榜”的海报布局。
尺寸：3:4。
配色：背景使用深蓝色 (#0052CC) 到黑色的渐变。
内容：
1. 标题：“今日收票战报” (Bold, White)。
2. 数据概览：一行两列，左边“总金额 3200W”，右边“服务客户 12家”。
3. 交易列表：设计一个类似“股票行情”的列表容器，列出 5 条数据（如 "张总 ... 50W ... 已到账"），每行带有绿色的“成功”图标。
4. 底部：醒目的黄色按钮样式文字 "月底额度告急，带票以此价为准"。
风格：科技感，数据可视化风格。
```

## 🎨 Canva 实操步骤 (SOP)

1.  **新建画布**: 在 Canva 创建自定义尺寸 `1080 x 1440 px`。
2.  **导入底图**: 使用 Week 2 Day 8 生成的 Midjourney 黑金/科技底图。
3.  **排版文字**: 
    - 按照 Claude 生成的布局位置，输入文字。
    - 金额数字使用 `DIN Alternate` 或 `Impact` 字体。
    - 标题使用 `思源黑体 Heavy`。
4.  **添加装饰**: 加上金币、光效、丝带等 PNG 素材（参考 `tools/ai_toolkit.md` 中的 Remove.bg 处理素材）。
5.  **导出**: 保存为 JPG (80% 质量，适合微信发送)。
